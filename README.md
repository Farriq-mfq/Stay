## CARA INSTALL

### LINUX

#### Server

1. install g++
   ```bash
   apt install g++
   ```
2. install nvm
   ```bash
   curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
   ```
3. install pnpm
   ```bash
   corepack enable
   ```
4. install build-essential
   ```bash
   apt install build-essential
   ```
